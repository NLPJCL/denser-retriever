## Install

We highly recommend using poetry to install Denser Retriever. If you don't have poetry installed, you can install it with the following command.

```bash
pip install poetry
```

Then, you can install Denser Retriever with the following command.

```bash
poetry add git+https://github.com/denser-org/denser-retriever.git#0.1.0
```

More details can be found the [DEVELOPMENT Document](https://github.com/denser-org/denser-retriever/blob/main/DEVELOPMENT.md).
